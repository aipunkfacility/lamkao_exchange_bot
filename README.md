# Telegram-бот для обменного сервиса "LamKao Exchange"

Этот проект представляет собой Telegram-бота, разработанного для автоматизации и управления операциями по обмену валют в Муйне, Вьетнам. Бот служит интерфейсом для клиентов, а также панелью управления для менеджера (администратора), упрощая процесс от подачи заявки до выдачи наличных.

## 🚀 Основные возможности

*   **Полный цикл обмена:** Автоматизация всего процесса: запрос клиента -> расчет суммы -> одобрение менеджером -> отправка реквизитов -> подтверждение оплаты -> генерация уникального кода для выдачи.
*   **Управление кассой:** Менеджер полностью контролирует поток заявок и одобряет их только при наличии достаточного количества наличных средств.
*   **Интерактивный чат с поддержкой:** Клиенты могут напрямую связаться с менеджером через бота, который выступает анонимным посредником. Менеджер управляет диалогом с помощью удобных инлайн-кнопок.
*   **Безопасная выдача:** Система генерации уникальных кодов для получения наличных исключает ошибки и риски мошенничества.
*   **Управление через Telegram:** Менеджер выполняет все действия (одобрение, ответ в чате, подтверждение) прямо в интерфейсе Telegram без необходимости в сторонних приложениях.
*   **Модульная структура:** Код организован по принципу разделения ответственностей (хэндлеры, клавиатуры, состояния), что упрощает его поддержку и расширение.

## 🛠 Технологический стек

*   **Python 3.11+**
*   **aiogram 3.x** — асинхронный фреймворк для создания Telegram-ботов.
*   **python-dotenv** — для управления переменными окружения.
*   **magic-filter** — для фильтрации событий в Aiogram 3.x.
*   **aiogram-sqlite-storage** — SQLite-хранилище для FSM состояний (заменяет Redis).
*   **SQLAlchemy 2.x (async)** — асинхронная работа с базой данных SQLite.
*   **FSM (Finite State Machine)** — для управления многошаговыми диалогами с пользователем.

## 📱 Команды бота

Бот поддерживает текстовые команды, которые можно вводить прямо в чате или выбирать из системного меню Telegram (кнопка Menu):

*   `/start` — 🏠 Главное меню
*   `/rates` — 💱 Актуальный курс валют
*   `/cancel` — ❌ Отменить текущее действие (прерывает любой процесс)
*   `/help` — ℹ️ Информация и контакты

### Особенности команд

- **`/cancel`** работает в любом состоянии и сбрасывает FSM
- **`/rates`** показывает актуальные курсы с форматированием (пробелы вместо запятых)
- **`/start`** всегда возвращает в главное меню и очищает состояние

## ⚙️ Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <ссылка_на_ваш_репозиторий>
cd <папка_проекта>
```

### 2. Создание и активация виртуального окружения

```bash
# Для Windows
python -m venv venv
venv\Scripts\activate

# Для macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

1.  Создайте файл `.env` в корневой папке проекта.
2.  Добавьте в него следующие переменные:

    ```env
    BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
    ADMIN_ID=ВАШ_ТЕЛЕГРАМ_ID
    SBER_CARD=НОМЕР_КАРТЫ_СБЕРА
    ```

*   `SBER_CARD` — номер карты Сбербанка для приема платежей (по умолчанию: "Реквизиты не указаны")

*   `BOT_TOKEN` можно получить у [@BotFather](https://t.me/BotFather).
*   `ADMIN_ID` можно узнать, написав боту [@getmyid_bot](https://t.me/getmyid_bot).

### 5. Запуск бота

```bash
python main.py
```

> **Примечание**: Бот использует SQLite для хранения состояний FSM (файл `fsm_storage.db`). Redis не требуется!

## 🔧 Разработка

*   Используйте `FSMContext` для хранения состояний пользователей
*   Все клавиатуры должны быть в `keyboards/keyboards.py`
*   Хэндлеры команд (`/start`, `/cancel` и т.д.) должны быть в начале файла
*   Используйте `StateFilter('*')` для глобальных команд, прерывающих любые состояния
*   Для доступа к базе данных используйте параметр `session: AsyncSession` в хэндлерах (предоставляется `DbSessionMiddleware`)
